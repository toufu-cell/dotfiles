# テンプレート索引

## SKILL.md テンプレート選択

| 発動パターン | テンプレートファイル | 使い分け |
|------------|-------------------|---------|
| コマンド起動型 | `skill-command.md` | `/xxx` で明示起動。SKILL.md は最小限 |
| 自動発動型 | `skill-auto-trigger.md` | キーワードで自動発動。発動条件 + 誤発動抑止を含む |
| 文脈判断型 | `skill-context.md` | 特定の作業文脈で適用。使用タイミングを含む |
| 常時ルール型 | `skill-always-on.md` | 常にルールとして適用。description に最重要ルールを含む |

## INSTRUCTIONS.md テンプレート選択

| 構造パターン | テンプレートファイル | 適するケース |
|------------|-------------------|------------|
| フェーズ分割型 | `instructions-phased.md` | 順序のあるワークフロー（例: obsidian-research, recipe） |
| クイックスタート+詳細型 | `instructions-quickstart.md` | 複数コマンド/機能を横断するスキル（例: security-review） |
| ガイドライン型 | `instructions-guideline.md` | 参照系のルール・規約集（例: design） |

## コマンドファイル

| テンプレートファイル | 用途 |
|-------------------|------|
| `command.md` | `/xxx` コマンドファイル |

## 選択フローチャート

```
要件を確認
  ├─ 「/xxx で起動」→ コマンド起動型
  ├─ 「〇〇で自動で」→ 自動発動型
  ├─ 「常にこのルールを」→ 常時ルール型
  ├─ 「〇〇の作業時に」→ 文脈判断型
  └─ 判定不能 → AskUserQuestion で確認
```
