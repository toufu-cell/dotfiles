# SKILL.md テンプレート: 文脈判断型

サイズ目安: ~300-625B

```markdown
---
name: {skill-name}
description: {スキルの目的を1文で}
---

## 使用タイミング

以下の作業時にこのスキルの INSTRUCTIONS.md を読み込む:
- {タイミング1}
- {タイミング2}
- {タイミング3}

詳細な手順は `INSTRUCTIONS.md` を参照（このスキルのディレクトリ内）。
```

## 置換ルール

| プレースホルダ | 説明 |
|-------------|------|
| `{skill-name}` | スキル名（kebab-case） |
| `{スキルの目的を1文で}` | 50文字以内で簡潔に |
| `{タイミングN}` | 適用すべき作業コンテキスト |

## 注意

- 使用タイミングは具体的な作業名（「React でのUI生成」等）で記述
- 抽象的な条件は避ける
