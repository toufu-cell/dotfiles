# Quurandot 問題攻略スキル

検索可能クイズ「Quurandot」の問題を体系的に解くためのガイドライン。
Quurandotは、インターネット検索・画像検索・AI活用などあらゆる手段を用いて解答する形式のクイズ企画。

## 企画概要

- **名前の由来**: オペラ『トゥーランドット』（Turandot）
- **形式**: 検索可能クイズ（あらゆる検索手段・ツールの使用が許可されている）
- **難易度**: Vol.0（チュートリアル）→ Vol.1〜4（段階的に難化）
- **各Volの構成**: 3問セット（1問目がやや易しめ、3問目が最難関の傾向）

## 問題の分類（6タイプ）

| タイプ | 説明 | 代表例 |
|--------|------|--------|
| テキスト推理型 | 文章から概念・法則を特定 | Q1-1（パーキンソンの凡俗法則） |
| 画像特定型 | 写真・画像から対象を特定 | Q0-2（ファルネーゼのアトラス）、Q3-2（馬房） |
| 形・図形解読型 | 幾何学的な形や回路図の意味を解読 | Q1-2（建築物の航空写真）、Q1-3（合成抵抗） |
| 暗号・符号解読型 | 暗号化された文字列を解読 | Q0-3（注音輸入法）、Q3-3（倉頡輸入法） |
| パターン・対応型 | 数字・日付・記号のパターンを発見 | Q2-1（覆面算）、Q2-2（ユールラッズ） |
| 複合型 | 複数の知識領域を横断 | Q4-2（エッフェル塔×凱旋門）、Q4-3（バベルの図書館） |

## 解法フロー（クイックリファレンス）

```
1. 問題の観察・分類
   → どのタイプか判断（上記6タイプ）

2. 手がかりの抽出
   → テキスト: キーワード、数字、固有名詞
   → 画像: 特徴的な形状、文字、色、構造
   → 記号: パターン、繰り返し、対応関係

3. 検索戦略の選択（→ search-strategies.md）
   → テキスト検索 / 画像検索 / 完全一致検索 / AI活用

4. 反復的な絞り込み
   → 検索結果から仮説を立てる → 検証 → 再検索

5. 解答の確信度チェック
   → 問題の全要素と矛盾なく整合するか確認
```
