# Quurandot よくある誤答パターンと注意点

過去の出題における典型的な誤答パターンと、解答時に気をつけるべきポイント。

## 1. 解答精度の問題

### 1.1 「惜しいが不正解」パターン

正解に近いが、微妙にずれた解答は誤答となる。

| 問題 | 正解 | よくある誤答 | 理由 |
|------|------|-------------|------|
| Q1-1 | 自転車置き場 | 駐車場 | 自転車置き場と駐車場は別物 |
| Q0-2 | いるか（座） | ファルネーゼのアトラス / 天球 | 「かたどっている部分」を問われているのに全体を回答 |
| Q3-2 | 具体的な場所名 | 馬房 | 「場所はどこ」に対して種類だけの回答は不十分 |
| Q4-2 | CARNOT | PETIET | 赤い点の位置と合致しない |

**教訓**:
- **問いに正確に答える**: 「何」「どこ」「誰」の区別を意識する
- **粒度を合わせる**: 「場所はどこ」→ 具体的な地名、「何をかたどった」→ 対象物の名前
- **図の詳細まで整合させる**: 赤い点の位置など、細かい要素も確認

### 1.2 上位概念・下位概念の取り違え

- 求められているのが具体名なのに抽象概念を答える
- 逆に、抽象概念を求められているのに具体例を答える

**例**:
- Q0-2: 「いるか」が正解なのに「天球儀」と答える（上位概念すぎる）

## 2. 「もっともらしいが違う」パターン

### 2.1 連想による誤答

問題のテーマから連想される単語だが、問題の構造と整合しない解答。

**例**（Q0-1）:
- 問題: Speranza, Sangue, ?（イタリア語の3つの謎）
- 誤答: Bianco, Fede, Pianura, Pace, Neve（イタリア国旗関連）
- 理由:
  - 国旗の解釈は多義的で、一意に定まる単語の組み合わせがない
  - 1→2→3 の数字の並びがトリコローレの並びと不一致

**教訓**:
- **一意性の確認**: 答えが「それしかない」と言えるか
- **全要素との整合**: 問題の全ての手がかりと矛盾しないか

### 2.2 部分正解で止まる

途中まで正しい方向性だが、最後まで掘り下げきれていない解答。

**例**:
- Q1-3: 「合成抵抗が1」→ 正しい方向だが「無限ラダー回路」と回答 → 全く異なる種類の回路
- Q4-2: 「エレベーター」→ エッフェル塔の特徴だが赤い点の位置と不整合

## 3. 問題の読み違え

### 3.1 問われていることの取り違え

- 「何を表しているか」→ 回路図そのものではなく、対応する概念
- 「この場所はどこ」→ 建物の種類ではなく具体的な場所
- 「共通する6文字」→ 概念ではなく文字列

### 3.2 注記・条件の見落とし

- 画像の加工方法の注記（「輪郭のみを残すように加工」等）
- 抵抗値の条件（「すべて1」等）
- 数字の桁数や文字数の指定

## 4. 不正解となる解答方法

### 4.1 ブルートフォース（総当たり）

可能な組み合わせをすべて送信する行為は「想定解に準ずる解答」とみなされない。

**例**（Q3-1）: かな2文字の全組み合わせを送信 → 不正解

### 4.2 こじつけ・ハック

問題の意図と異なる方法で答えを「当てる」行為。

**注意**: 複数回解答を送信した場合、最後に送信された内容で正誤判定が行われる。

## 5. 解答前のセルフチェックリスト

### 必須チェック

- [ ] 問われていること（何/どこ/誰/いくつ）を正確に理解しているか
- [ ] 問題の **全ての** 手がかり（テキスト、画像、数字、色、位置など）と矛盾しないか
- [ ] 答えが一意に特定できているか（「それしかない」と言えるか）
- [ ] 具体性が十分か（抽象的すぎないか / 具体的すぎないか）
- [ ] 別の情報源で裏取りしたか

### 注意チェック

- [ ] 連想だけで答えていないか（検索で確認したか）
- [ ] 正解の近くで止まっていないか（もう一段掘り下げる必要はないか）
- [ ] 問題の注記や条件を見落としていないか
- [ ] 図の細部（赤い点の位置、方向、数字など）まで整合しているか
- [ ] AI の回答をそのまま信じていないか（裏取りしたか）

## 6. 別解として認められるケース

完全な正解でなくても、以下の条件を満たす場合は別解として正解になることがある:

- **想定解の言い換え**: 同じ概念を別の表現で述べている（例: Q0-1の「suo stesso nome」= Turandotの言い換え）
- **出題の形式に合わせた再変換**: 問題と同じ変換方式で解答している（例: Q0-3で玉山を注音コードで回答）
- **準拠する出典が存在する**: 別の信頼できる出典に基づく解答（例: Q0-1の「Amore」はタイのショーに基づく）

## 7. Vol別の傾向と対策

| Vol | 傾向 | 注意点 |
|-----|------|--------|
| 0 | チュートリアル的。検索の基本を問う | 油断せず正確に答える |
| 1 | 言い換え検索、画像検索の応用 | 検索キーワードの工夫が鍵 |
| 2 | 暗号・符号、パターン認識 | 文字体系・入力方式の知識が有利 |
| 3 | 視覚的な謎解き、場所特定 | 画像の詳細な観察と多角的な検索 |
| 4 | 複合的、専門的な知識を横断 | 粘り強い調査と要素の統合が必要 |
