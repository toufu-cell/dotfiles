---

## 使用タイミング

- プロジェクトのコードベース整理・クリーンアップ時
- 依存パッケージの棚卸し時
- リファクタリング前の不要コード除去時
- `/refactor-clean` コマンド実行時

## 使用しないタイミング

- 機能開発中（開発完了後に実行する）
- 本番デプロイ直前
- テストカバレッジが不十分な場合
- 変更差分のレビュー → simplify スキルを使う

## 対応言語（v1）

| 言語 | 主要ツール |
|------|-----------|
| JavaScript/TypeScript | knip |
| Python | vulture |
| その他 | Grep（候補ヒントのみ、自動削除不可） |

詳細な手順は `INSTRUCTIONS.md` を参照（このスキルのディレクトリ内）。
