# Claude Code ツール使用ルール

## 専用ツールの優先使用（必須）

ファイル操作は専用ツール（Read, Edit, Write）を優先し、並列実行可能な操作は並列で実行する。
大きな変更の前に計画を提示すること。

### Bash コマンドの代わりに専用ツールを使う

以下のコマンドを Bash ツールで実行してはならない。対応する専用ツールを必ず使用すること:

| やりたいこと | 禁止（Bash） | 使うべきツール |
|---|---|---|
| ファイルを読む | `cat`, `head`, `tail`, `less` | **Read** |
| ファイルを検索する | `find`, `ls -R` | **Glob** |
| ファイル内容を検索する | `grep`, `rg`, `ag` | **Grep** |
| ファイルを編集する | `sed`, `awk`, `perl -i` | **Edit** |
| ファイルを作成する | `echo >`, `cat <<EOF >`, `tee` | **Write** |
| テキストを出力する | `echo`, `printf` | 直接テキスト出力 |

**Bash ツールは、上記に該当しないシステムコマンドやシェル操作（`git`, `npm`, `docker`, プロセス操作等）にのみ使用する。**

## タスク管理

- 複雑なタスクはTodoWriteで管理
- タスクの進捗をリアルタイムで更新
- 完了したタスクは即座にマーク
