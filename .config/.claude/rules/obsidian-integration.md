# Obsidian Vault 統合

Claude Codeセッションでは、以下のObsidian vaultを参照できる:

- **Vaultパス**: `${HOME}/Library/Mobile Documents/iCloud~md~obsidian/Documents/Obsidian_Vault`
- **主要ディレクトリ**: `IT用語集/`, `参考論文/`, `Clippings/`

## 調査・リサーチ

「調べて」「まとめて」「〇〇とは」等の調査依頼には、
`~/.claude/skills/obsidian-research/` の INSTRUCTIONS.md に従って実行すること。
発動条件・保存先・誤発動抑止・機密除外ルールはスキル側で定義されている。
